# Security Training Management Application

A comprehensive Electron-based desktop application designed to manage student records, training processes, and administrative workflows for security training institutions.

## 🚀 Features

- Student enrollment and management system  
- Document tracking and organization  
- Installment-based payment system  
- Automated PDF receipt generation  
- Excel form generation and reporting  
- Notification system for administrative workflows  
- Fully localized and user-friendly interface  

## 🛠️ Tech Stack

- Electron.js  
- JavaScript  
- MariaDB
